# 🤖 KI-Firma

Eine vollständige KI-Firma mit CEO und Team-Agenten.

## Agenten

| Agent | Aufgabe |
|-------|---------|
| 👔 CEO | Koordiniert alles, delegiert an das Team |
| 👨‍💻 Developer | Baut Apps und schreibt Code |
| 🔍 Researcher | Recherchiert Themen und schreibt Berichte |
| 📣 Marketing | Erstellt Texte, Posts und Kampagnen |
| 📊 Analyst | Macht Business-Analysen und Strategien |

## Setup

### Environment Variables (in Render eintragen)

| Variable | Wert |
|----------|------|
| `ANTHROPIC_API_KEY` | Dein Claude API Key |

## Benutzung

Öffne die App-URL und gib eine Aufgabe ein, z.B.:
- "Baue eine einfache To-Do App in Python"
- "Schreib einen LinkedIn-Post über KI-Trends"
- "Analysiere den KI-Markt 2025"
- "Erstelle einen Business-Plan für eine KI-Firma"
